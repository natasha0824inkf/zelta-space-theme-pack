# Zelta Slack Theme

A dark, cosmic Slack theme inspired by [zelta.space](https://zelta.space) — navy depths with gold accents.

## Color Palette

| Role | Hex | Preview |
|------|-----|---------|
| Column Background | `#1C2535` | Deep navy |
| Menu Hover | `#243044` | Lifted navy |
| Active Item | `#C9A227` | Brand gold |
| Active Item Text | `#1C2535` | Dark navy |
| Hover Item | `#243044` | Lifted navy |
| Text | `#E8EDF5` | Soft white |
| Active Presence | `#C9A227` | Brand gold |
| Mention Badge | `#D4580A` | Amber alert |

## Installation

1. Open Slack
2. Go to **Preferences → Themes**
3. Scroll down and click **Create a custom theme** (or paste into the custom theme box)
4. Paste the theme string below:

```
#1C2535,#243044,#C9A227,#1C2535,#243044,#E8EDF5,#C9A227,#D4580A
```

5. Hit Enter — done!
